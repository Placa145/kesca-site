# KESCA — Kitchen Exhaust System Cleaners Association

The official site for KESCA, a Canadian non-profit that certifies kitchen exhaust cleaning contractors and commercial kitchens to **NFPA 96** fire-prevention standards.

Founded in **2004** by Rob Sweet — in the indoor air quality trade since 1988 — to fix an industry that was selling stickers instead of doing the work.

## Stack

Plain HTML/CSS/JS. No build step. Single `index.html`, inline CSS, inline SVG illustrations. Static assets in `public/`. Deployed on Vercel.

## Local dev

```bash
npx serve . -l 5057
# then visit http://localhost:5057
```
